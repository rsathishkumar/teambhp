AMP Carousel
-------------

INTRODUCTION
This modules extends AMP module and AMP Theme to support the amp-carousel component.

Amp Carousel: https://amp.dev/documentation/examples/components/amp-carousel/

It exposes a configurable field formatter for image fields.

CONFIGURATION
* Layouts: slides, carousel
* Types: responsive, fixed-height
* Autoplay with delay

REQUIREMENTS
This module requires AMP module and AMP Theme to be installed.

INSTALLATION
Place inside sites/all/modules directory & profit
